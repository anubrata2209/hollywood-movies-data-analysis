# 🎬 Hollywood Movies Data Analysis

This project performs exploratory data analysis (EDA) on a Hollywood movies dataset using Python.  
The goal is to analyze movie budgets, genres, audience ratings, and Rotten Tomatoes scores.

## 📂 Dataset Features
The dataset contains the following columns:
- Movie Name
- Genre
- Budget (Million $)
- Audience Rating (%)
- Rotten Tomatoes Rating (%)
- Year of Release
## 🛠 Technologies Used
- Python
- Pandas
- Jupyter Notebook
- Data Visualization
## 📊 Analysis Performed
The project explores:
- Movie budget distribution  
- Audience rating trends  
- Rotten Tomatoes score comparison  
- Movie genre analysis  
- Year-wise movie release patterns
